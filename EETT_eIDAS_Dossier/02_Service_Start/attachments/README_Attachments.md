# Attachments – Service Start Application (Folder 02)

This directory must contain **all documentary evidence** referenced in `02_Service_Start_Application.md` and required by:

* EETT Article 6.3 (Annex 4, Parts C–E)  
* eIDAS Regulation Art 20 & 24  
* ETSI EN 319 411-1 §5 & §7 (financial, technical, organisational resources)

## 2. Naming Convention

```
<Prefix>_<ShortDescription>_<YYYYMMDD>.<ext>
```

Examples  
`Financial_Statements_2024.pdf`  
`Insurance_Policy_Professional_Liability_20250201.pdf`
