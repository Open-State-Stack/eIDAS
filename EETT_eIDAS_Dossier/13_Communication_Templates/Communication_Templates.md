## Communication Templates – Quick Reference

**Last Review:** 2025-11-15 | **Reviewer:** Senior Compliance Officer

### Key Assets
- `13_EETT_Notification_Templates_GR.md` – Master pack of nine bilingual templates covering change notices, incident (initial/final), termination, CAR, RFI responses, GDPR breaches, CA compromise, and cross-border incidents (Template 9 now available in EN).
- `13_EETT_Communication_Templates_GR.pdf` – Signed PDF version for submission.
- `EXECUTIVE_SUMMARY_FOR_STAKEHOLDERS.md` & `FINAL_QA_VERIFICATION_REPORT.md` – QA evidence and stakeholder briefing.
- `14_NTL_Publication_Datasheet.md` – Reference for service endpoints/URLs mentioned in notifications.

### When to Use Each Template
| Scenario | Template | Deadline | Cross-Refs |
|----------|----------|----------|------------|
| Corporate change | Template 1 | 7 calendar days | Doc 02 (Registry) |
| Security incident ≥ Level 3 (initial) | Template 2 | ≤24 h | Doc 07b (Incident Response) |
| Security incident final report | Template 3 | Per EETT deadline | Doc 07d (RCA) |
| Planned termination | Template 4 | ≥3 months | Doc 09 |
| Annual CAR submission | Template 5 | yearly | Doc 03 |
| EETT info request response | Template 6 | 5–25 business days | Doc 19 |
| GDPR/personal data breach | Template 7 | 24 h (EETT) / 72 h (HDPA) | Doc 08 / Doc 12 |
| CA private key compromise | Template 8 | <1 hour | Doc 07b §7 |
| Cross-border incident (EETT + ENISA + other MS) | Template 9 (GR & EN) | 24 h | Doc 08 §7 |

### Distribution & Contact Lists
- **Primary regulator recipients:** Use the addresses maintained in `08_User_Notification_Plan_GR_EN.md` §10 to ensure hotline/email parity between Docs 08 and 13.
- **Cross-border distribution:** For Template 9, pull the supervisory contacts per Member State from `MASTER_CONTACT_LIST.md` (Doc 01) and mirror any updates made there; this prevents drift between notification and recordkeeping teams.
- **Internal approval flow:** Draft → Compliance review → CEO/DPO sign-off → Dispatch via secure email with QSeal signature.

### Outstanding Actions Before Submission
1. Replace all `[PLACEHOLDERS]` (names, phone numbers, emails, URLs, dates) with the final master data extracted from Docs 01 & 08.
2. Attach the bilingual Template 9 (EN) when notifying ENISA or foreign supervisors; retain the sent version in Doc 12 evidence archives.
3. Keep the distribution alignment log (Doc 08 §10) updated whenever contact lists change—Templates in Doc 13 should always reference the latest list.
